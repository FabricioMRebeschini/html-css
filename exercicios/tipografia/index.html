Anatonia dos tipos

A letra 'x' minuscula é um ponto de partida para medidas de todas as outras letras. A altura x!

Tem também altura das maiúsculas e altura ascendente(vazamento pra cima) e descendente vazamento pra baixo A altura total é chama de CORPO

Uma letra pode ter Serifa, Hastes, Filete, Arco,barriga,braço,perna,pé,espinha(s),olho, orelha,calda,vertice, terminal e esporão.

Glifo é uma letra. A B C D = glifos

conjunto de glifos,caractere ou letras = fonte

Existem as famílias tipograficas que é as variações. negrito, light, normal.

Categoria de fontes

serifada com serifada = texto longo sans serife = sem serifa = texto curto branco no fundo e preto na letra para texto grande monoespaçada = todas as letras com mesmo espaço handwriting = sensação feita a mão display = fontes comemorativas
